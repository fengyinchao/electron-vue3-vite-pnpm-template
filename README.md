# electron-vue3-vite-pnpm-template

基于 electron-vue3-vite-pnpm 开发的模板

# 特色

## 前沿技术栈

- pnpm 管理依赖
- vite 快速构建
- vue3 组合式 API
- typescript

## 流行的开发规范

- eslint
- prettier
- husky
- commitlint
- editorconfig
- changeset

# 开发

## 安装依赖

```bash
pnpm install
```

## 运行

```bash
pnpm dev
```

## 调试

vscode 切到 运行和调试 按钮，选择 调试主进程-Main

## 打包

```bash
pnpm package
```

## 制作安装包

```bash
pnpm make
```
